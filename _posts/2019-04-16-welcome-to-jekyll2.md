This is just a welcome page to test out bitbrain's Jekyll-dash theme. All the other posts will be fetched from `_posts` directory. The most common way to edit this site is to run `jekyll serve`, which launches a web server and auto-regenerates this site when a file is updated.

To add new posts, simply add a file in the `_posts` directory that follows the convention `YYYY-MM-DD-name-of-post.ext` and includes the necessary front matter. Take a look at the source for this post to get an idea about how it works.
